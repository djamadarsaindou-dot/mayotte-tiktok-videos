🎬 Mayotte TikTok

Application personnelle qui génère et publie automatiquement des vidéos TikTok éducatives sur Mayotte (légendes, traditions, faune, actualités).

Compte TikTok : @mister_decouverte
Auteur : Djama Darsaindou (Mayotte, France)
Code source : github.com/djamadarsaindou-dot/mayotte-tiktok-videos

🛠 Fonctionnement

  1. Toutes les 6h, un script Python génère une vidéo verticale TikTok (1080×1920)
  2. Le script utilise une IA (Mistral) pour écrire un texte éducatif sur Mayotte
  3. Une voix de synthèse (Coqui XTTS) narre le texte
  4. Des images IA (Pollinations) et stock (Pexels) illustrent le récit
  5. FFmpeg assemble la vidéo finale avec sous-titres karaoké
  6. La vidéo est envoyée dans la boîte de réception TikTok de l'auteur pour validation

Cette application n'est utilisée que par son auteur pour publier sur son propre compte TikTok personnel. Aucune donnée d'utilisateur tiers n'est collectée.
