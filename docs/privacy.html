Politique de confidentialité

Dernière mise à jour : 20 mai 2026

1. Qui sommes-nous

« Mayotte TikTok » est une application personnelle développée et utilisée exclusivement par Djama Darsaindou (Mayotte, France) pour publier des vidéos sur son propre compte TikTok @mister_decouverte. Cette application n'est pas distribuée à des tiers et n'a pas d'utilisateur final autre que son auteur.

2. Données collectées

L'application n'enregistre aucune donnée personnelle d'utilisateurs tiers. Les seules données techniques manipulées sont :

  • Les tokens d'accès TikTok de l'auteur (stockés localement dans un fichier .env chiffré, jamais envoyés à des tiers)
  • Les vidéos générées (stockées localement sur l'ordinateur de l'auteur)
  • Les compteurs de rotation thématique (fichier texte local)

3. Utilisation de l'API TikTok

L'application utilise l'API TikTok Content Posting pour :

  • Uploader des vidéos sur le compte TikTok de l'auteur (scope video.upload ou video.publish)
  • Récupérer le profil basique de l'auteur pour valider l'authentification (scope user.info.basic)

Aucune donnée d'autres utilisateurs TikTok n'est consultée ni stockée.

4. Partage des données

Aucune donnée n'est partagée avec des tiers. Les tokens d'accès TikTok sont conservés localement et ne quittent jamais la machine de l'auteur.

5. Sécurité

Les tokens d'accès sont stockés dans un fichier .env exclu du versioning Git (via .gitignore) et accessible uniquement à l'auteur sur sa machine personnelle.

6. Vos droits

L'application n'ayant pas d'utilisateur final autre que son auteur, les questions de droit d'accès / rectification / suppression ne s'appliquent pas à des tiers. L'auteur peut révoquer ses propres tokens à tout moment depuis les paramètres TikTok.

7. Contact

Pour toute question : GitHub issues
